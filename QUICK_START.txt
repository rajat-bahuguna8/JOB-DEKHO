  TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════

Problem: Port 5000 already in use
Solution: Edit app.py line 58, change port to 5001

Problem: Database error
Solution: Delete job_dekho.db and run: python init_db.py

Problem: Module not found
Solution: Run: python -m pip install -r requirements.txt

Problem: Template not found
Solution: Check that templates/ folder exists with all files
